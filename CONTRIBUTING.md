# Contribuição

Obrigado por ajudar a melhorar o RepoHealth.

## Boas primeiras contribuições

- Melhorar os templates criados por `repohealth fix`
- Adicionar checks para mais ecossistemas
- Melhorar exemplos do README
- Adicionar testes para casos de borda
- Melhorar a interface gráfica local

## Desenvolvimento

1. Clone o repositório.
2. Use Node.js 18 ou mais recente.
3. Rode a CLI localmente:

```bash
node ./src/cli.mjs scan
```

4. Abra a interface local:

```bash
node ./src/cli.mjs ui
```

5. Rode os testes:

```bash
npm test
```

## Pull requests

Mantenha pull requests focados. Explique o problema, a solução e como você testou a mudança.
