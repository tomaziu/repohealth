# Changelog

Todas as mudanças relevantes do RepoHealth serão documentadas neste arquivo.

## 0.3.0

- Relatório Markdown com prioridades, checks e prompt para IA.
- Comando `repohealth report`.
- Download de relatório pela interface gráfica.

## 0.2.0

- Interface gráfica local com `repohealth ui`.
- Saída da CLI em português brasileiro por padrão.
- Suporte a `--lang pt-BR|en`.

## 0.1.0

- CLI inicial com `scan`, `fix`, saída JSON e nota mínima.
